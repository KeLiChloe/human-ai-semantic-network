Semantic Network Explorer

Choose a network from the left sidebar.

Node
Each node represents one participant. Humans are shown as anonymized contributor IDs; GenAI models keep their names.
Edge
Semantic similarity between two explanations. An edge is drawn when their embedding cosine similarity meets or exceeds the within-group threshold shown on each network page.

How to explore

  • Hover a node to read the contributor ID and full theoretical explanation.
  • Click a node to pin the detail panel; press Esc to clear.
  • Search by contributor ID using the toolbar on each network page.
